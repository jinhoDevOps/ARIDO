### 1. 일반적인 `passwd` 명령어 사용법
일반 사용자가 자신의 비밀번호를 변경하고자 할 때 사용합니다. 

```bash
passwd
```

이 명령을 실행하면 현재 비밀번호를 묻는 메시지가 나타납니다. 현재 비밀번호를 정확히 입력한 후, 새 비밀번호를 두 번 입력하여 변경을 완료합니다.

### 2. 관리자 권한으로 강제 비밀번호 변경
루트 또는 관리자 권한으로 다른 사용자의 비밀번호를 강제로 변경할 수 있습니다. 루트 사용자로 로그인한 상태에서 다음과 같이 입력합니다:

```bash
passwd [사용자 이름]
```

여기서 `[사용자 이름]`에는 비밀번호를 변경하고자 하는 사용자의 이름을 입력합니다. 
이 명령을 실행하면 새 비밀번호를 두 번 입력하라는 메시지가 나타납니다. 새 비밀번호를 입력하면 해당 사용자의 비밀번호가 갱신됩니다.

### 리눅스에서 `su` 명령어를 사용하여 변경된 비밀번호 확인하기

-  **`su` 명령어 실행**: 변경된 비밀번호를 테스트하고자 하는 사용자 계정으로 전환하기 위해 `su` 명령어를 사용합니다. 명령어는 다음과 같습니다:
   ```bash
   su - [사용자 이름]
   ```
   여기서 `[사용자 이름]`에는 테스트하고자 하는 계정의 사용자 이름을 입력합니다.

-  **비밀번호 입력**: 시스템이 해당 사용자 계정의 비밀번호를 요청할 것입니다. 변경한 새 비밀번호를 입력합니다.

- **로그인 성공 확인**: 새 비밀번호가 정확하다면, 사용자 계정으로 성공적으로 전환됩니다. 이를 통해 비밀번호 변경이 정상적으로 이루어졌음을 확인할 수 있습니다.
